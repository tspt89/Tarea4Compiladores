Instrucciones para su ejecución:
	-Para ejecutarlo, simplemente se tiene que correr el archivo execute.sh. (Donde se compila el flex, el bison, y al final se integran ambas librerias).
		Para ejecutarlo puede funcionar:
			$sh execute.sh
			$./execute.sh
			(En caso necesario, utilizar sudo).
	-Se ingresa la operación correspondiente (suma, resta, multiplicación, división, módulo y potencia).
	-Al final de la operación, se termina con un $ para indicar el final de la misma y se pueda mostrar el valor correspondiente de cada operación.
	-En caso de terminar la ejecución, para pararlo, se tiene que presionar: Ctr + C

Estos serían unos ejemplos de operaciones al escribir en la terminal:
Suma: 			22+2020$
Resta:			1024-64$
Multiplicación:	3*7$
División:		10/5$
Potencia:		2^5$
Modulo:			128%100$
